JUST 3 QUICK STEPS

There is zero installation required. Just extract the ZIP archive, and you are already up and running.

In a nutshell, installing Grav is a matter of unzipping the file to your web server.
Content is just simple markdown files in folders, with no complicated databases to deal with.
A powerful API and extensive hooks allow for plugins to extend Grav, such as the Admin Plugin.
